jQuery rsCheckbox plugin

Replace checkboxes with stars. Or any image, or anything, the plugin insert a span (by default, you can change it) the rest it's css.

How it works

Load jquery, load the plugin and call it, really simple

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="jquery.rscheckbox.js"></script>
<script>
$('input[type="checkbox"]').rsCheckbox();
</script>

Options (default)

{
    element : 'span', // Element to insert
    classNorm : 'favorites', // Class applied to element
    classHigh : 'selected', // Class applied when checked
    hideCheck : true // Show or not the real checkbox
}
